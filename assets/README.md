# Assets Folder

Place your game assets here:

## Required Assets:

### Main Assets (in /assets/)
- `pitch.png` - Football pitch background (1280x720)
- `ball_default.png` - Default ball sprite
- `goal.png` - Goal sprite (left side)
- `wall.png` - Wall sprite (right side)

### Ball Skins (in /assets/balls/)
- `golden.png` - Golden Ball
- `fire.png` - Fireball
- `steel.png` - Steel Ball
- `ghost.png` - Ghost Ball
- `spark.png` - Spark Ball
- `shadow.png` - Shadow Ball (coming soon)
- `cosmic.png` - Cosmic Ball (coming soon)
- `plasma.png` - Plasma Ball (coming soon)
- `titan.png` - Titan Ball (coming soon)

## Asset Guidelines:
- Use PNG format with transparency where needed
- Recommended ball size: 64x64 pixels
- Keep file sizes optimized for web
